See the [Getting Started](../guides/gettting-started.md) guide for information on how to get setup with the Beamable CLI.

All of the Beam CLI commands are available in the left navigation bar. This table links directly to some of the most common commands.

| Command                      | Description                                                             |
| :--------------------------- | :---------------------------------------------------------------------- |
| [Init](cli-command-reference/cli-init.md)         | `beam init` will create a Beamable project in the CLI                   |
| [Config](cli-command-reference/cli-config.md)     | `beam config` helps you verify your current Beamable project in the CLI |
| [Login](cli-command-reference/cli-login.md)       | `beam login` helps you manage access tokens for use in the CLI          |
| [Project](cli-command-reference/cli-project.md)   | `beam project` helps you create and use Standalone Microservices        |
| [Services](cli-command-reference/cli-services.md) | `beam services` help you and deploy Standalone Microservices            |
