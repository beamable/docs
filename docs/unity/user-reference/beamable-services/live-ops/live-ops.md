# Live Ops

Beamable offers powerful and flexible solutions for communicating with your players and allowing players to interact with your game.

## Features

| Feature | Details |
|---------|---------|
| [Events](events-overview.md) | Create engaging time-limited competitions and activities for players. Events provide a powerful way to drive player engagement through time-limited challenges, competitions, and special activities that create urgency and community participation. |
| [Announcements](announcements-overview.md) | Create and manage in-game announcements for players. Announcements provide a powerful way to communicate with players about updates, events, news, and other important information directly within your game. |
| [Notifications](notifications-overview.md) | Message players through a publish/subscribe pattern system. Notifications provide a robust messaging system that enables communication between different parts of your game through real-time event handling. |
| [Mail](mail-overview.md) | Create and manage in-game mail messaging systems for players. Mail provides a robust messaging system that enables players to send and receive persistent messages with attachments and rewards. |
