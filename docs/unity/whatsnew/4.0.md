# Version 4.0

This is the release notes for the The Unity SDK version 4.0


## Highlights

### Upgrade to CLI 6.x

This version of the SDK upgrades to use CLI 6.1, which means your Unity projects will start to benefit from the Microservice improvements in CLI 6.1. Notably, this release includes the new logging capabilities, which means when you publish a Microservice, you will get a whole new log experience in the Portal. 

Please note, once you update, you will not be able to see the old logs. We are not merging old and new log streams. If you need to see old log data (while it is still available), you can downgrade your project back to Unity Beamable SDK 3.x. 

### Official Unity 6.2 Support

The Beamable Unity SDK 4.0 adds official support for Unity 6.2 release. 

## Other Changes

Read our developer changelog [https://changes.beamable.com/?unity](https://changes.beamable.com/?unity)

## Upgrading

There are no known breaking changes for upgrading the Unity Beamable SDK from 3.x to 4.0. However, please review the [CLI 6.x upgrading guide](../../cli/guides/upgrading.md#from-54x-to-600), as it may be relevant.
